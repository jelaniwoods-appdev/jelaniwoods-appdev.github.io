Three ways:

## 1: Use Quotes

```
title: "How to escape colon: in Front matter YAML"
```

## 2: Use HTML entity

```
title: How to escape colon&#58; in Front matter YAML
```

## 3: Use multi-line title

```
title: >
  How to escape colon: in Front matter YAML
```
